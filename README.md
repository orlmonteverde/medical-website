# Generic medical website

Web responsive of medical purpose with **ES6**, **Vue.js** and **Sass**.

## Build Setup

```bash
# install dependencies
npm install

# Watch the changes and build the files automatically
npm run dev

# Build files for production with minification
npm run build
```
